# AZ-204 랩 매핑

이 과정, AZ-020: AWS 개발자용 Microsoft Azure 솔루션은 숙련된 AWS 개발자를 대상으로 하는 3일 과정에 적합하도록 조정된 AZ-204 과정 자료의 하위 집합입니다.

AZ-020 랩은 AZ-204 랩의 하위 집합으로서 [AZ-204 랩 리포지토리](https://github.com/MicrosoftLearning/AZ-204KO-DevelopingSolutionsforMicrosoftAzure)에서 가져온 것입니다.

## AZ-204 랩에 대한 AZ-020 모듈 매핑

이 과정에서 사용되는 AZ-204 랩의 하위 집합 10개는 다음 모듈별로 매핑되었습니다.

| AZ-020 랩 # | AZ-204 랩 # | 항목 |
| --- | --- | --- |
| 1 | 1 | App Services |
| 2 | 2 | Azure Functions |
| 3 | 3 | Azure Blob Storage |
| 4 | 4 | Azure Cosmos DB |
| 5 | *랩 없음* | ARM(Azure Resource Manager) 템플릿 |
| 6 | 6 | 인증 및 ID |
| 7 | 7 | 클라우드 서비스\[s\] 보안 |
| 8 | 8 | API Management |
| 9* | 10 | 이벤트 기반 메시징 |
| 10* | 11 | 비동기 메시징 |
| 11* | 12 | Azure Monitor |

**\*** *AZ-020 모듈 번호는 AZ-204 랩 번호와 다릅니다.*
