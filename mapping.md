# AZ-020부터 AZ-204까지의 랩 매핑

이 과정(AZ-020)에서는 다음에 대해 다룹니다. AWS 개발자를 위한 Microsoft Azure 솔루션은 숙련된 AWS 개발자를 위한 3일 간의 과정에 맞게 조정된 AZ-204 과정 자료의 하위 집합입니다.

AZ-020 랩은 AZ-204 랩의 하위 집합이며 [AZ-204 랩 리포지토리](https://github.com/MicrosoftLearning/AZ-204KO-DevelopingSolutionsforMicrosoftAzure)에서 직접 가져옵니다.

> 참고: 이 AZ-020 모듈 5에는 랩이 없습니다. 데모는 과정 콘텐츠를 참조하세요.

## AZ-204 랩에 대한 AZ-020 모듈 매핑

이 과정에 사용된 10개의 AZ-204 랩 중 일부는 모듈별로 다음과 같이 매핑됩니다.

| AZ-020 랩 # | AZ-204 랩 # | 토픽 |
| --- | --- | --- |
| 1 | 1 | App Services |
| 2 | 2 | Azure Functions |
| 3 | 3 | Azure Blob Storage |
| 4 | 4 | Azure Cosmos DB |
| 5(*랩 없음*) | - | ARM(Azure Resource Manager) 템플릿 |
| 6 | 6 | 인증 및 ID |
| 7 | 7 | 클라우드 서비스\[s\] 보안 |
| 8 | 8 | API Management |
| 9 | 9 | 이벤트 기반 메시지 |
| 10 | 10 | 비동기 메시징 |
| 11 | 11 | Azure Monitor |
